# CEHRMS – Centralized Electronic Health Record Management System  
A full-stack ASP.NET application designed to digitalize hospital workflows, enable seamless patient–doctor interactions, and securely manage medical records.

## Overview  
CEHRMS (Centralized Electronic Health Record Management System) is a web-based platform that streamlines healthcare operations such as appointment booking, patient record management, doctor schedules, billing, and more.

The system supports four user modules:
- **Patient**
- **Doctor**
- **Admin**
- **Receptionist**

It ensures secure login, role-based access, and complete digital health record maintenance using a unique **Health ID** generated at registration.

---

## Tech Stack  
**Frontend:** ASP.NET WebForms, HTML, CSS, Bootstrap  
**Backend:** C#  
**Database:** SQL Server  
**Architecture:** 3-Tier Architecture  
**Additional Features:** Email OTP Verification, Dynamic Health Card Generation, Encrypted Credentials

---

## Key Features  

### Patient Module  
- Register with email verification  
- Auto-generated **Health Card** with unique Health ID  
- Book appointments (hospital → department → doctor → date/time)  
- View past and upcoming appointments  
- View/download pathology reports  
- Manage profile & change password  
- View prescribed medicines  
- Bills & payment history  
- Submit feedback / ratings  

### Doctor Module  
- View patient list assigned to the doctor  
- Access complete patient history and medical reports  
- Update prescriptions and upload reports  
- Apply for leave  
- Manage daily appointment schedule  

### Admin Module  
- Manage doctors (CRUD operations)  
- Approve/reject doctor leave  
- Manage appointments  
- Manage hospitals, departments  
- Oversee patient records  
- Can reply to feedback or grievance   

### Receptionist Module  
- Manage patient appointments  
- Book slots  
- Update visit details  
- Generate invoices
